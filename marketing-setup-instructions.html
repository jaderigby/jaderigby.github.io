Marketing Setup

Instructions:

  • Download bacon:
    • Move it to the Documents folder
    • Unzip it
    • Rename it to just bacon
  • Run the bacon setup script: 
    • python ~/Documents/bacon/setup.py --exec && cd ~/Documents/bacon && setup.py && source ~/.zshrc
  • Get gitlab all set up — register and then generate and add an ssh key: 
    • mkdir ~/.ssh && cd ~/.ssh && ssh-keygen -t ed25519 -C "your_email@example.com" && ssh-add -K id_ed25519
  • Clone web repo and set it up: 
    • cd ~/Documents/bacon-bits && git clone git@gitlab.com:PassiveLogic/Marketing/hugo-website-utility.git && bacon alias && bacon set
  • Install Visual Studio Code: 
    • brew install --cask visual-studio-code
